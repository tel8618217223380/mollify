Instructions

Folder path is the location of the folder in the server filesystem, and this is not visible to users.

Folder path can be given either as absolute or as relative.

Absolute path is a path that points to the same location on the file system regardless of the working directory, and is usually written in reference to a root directory. An example of a absolute path: /var/www/files/images/ or C:\files\images\

Relative path in Mollify is relative to the Mollify backend folder. An example of a relative path: ../../files/images/
